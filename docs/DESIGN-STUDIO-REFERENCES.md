# Design Studio — Editorial References

*Collected from Dustin's mood board session, Feb 10 2026*

## Core Principle
The Design Studio should feel like opening a magazine feature spread, not clicking into an app modal. Every element earns its space.

## Reference Spreads

### 1. "Portrait of a Lady" — DA MAN
- Full-bleed dark hero image, massive white typography
- Split light/dark energy

### 2. "Potere in Vista" — Italian Editorial
- Dominant hero left, supporting images scattered right
- Item callouts alongside body text (like material specs)
- Bold subheads anchoring sections

### 3. "INTO THE DARK" — DA MAN
- Name split across spread, photo cutting through letterforms
- Pure black background, minimal body copy
- Title IS the design element

### 4. "Morgan Postins" — Skate Editorial
- Color blocking as structure (gold band creates zones)
- Itemized product callouts with bold labels on right
- Monogram/initials as graphic element
- Mixed B&W and color photography

### 5. "Linda Wisdom" — Photography Feature
- Full-bleed image left, clean serif right
- Hierarchy: small category → massive name → intro → body
- Effortless whitespace, nothing fights for attention

### 6. Drop Cap "A" — Fashion Editorial
- Giant single letterform as design element
- Tension between heavy black type and airy pastel image
- Left page = pure typography, right page = pure image

### 7. "Unearth" — Photography Feature ⭐ PRIMARY REFERENCE
- Small colored category label ("PHOTOGRAPHY")
- Massive serif title, mixed case
- Italic subtitle/lede in different voice
- Thin horizontal rule separator
- Two-column body text
- Small inset image within text column
- Full-bleed hero on left page

### 8. "LINDER" — Art Feature

### 9. SCREEN Posters — Bulldog Clips

- Gallery/review wall feeling
- → USE FOR: Lookbook card "pinned" state indicator

### 10. Positano Detail Grid — Travel Book
- One dominant hero, then grid of 3 supporting detail shots

- Multiple scales of imagery working together
- → USE FOR: Iteration results grid below main visualization

## Design System Decisions

### Typography
- **Titles:** Georgia / Times New Roman / serif — mixed case, not all-caps for editorial feel
- **Category labels:** 10-11px, uppercase, wide tracking, accent colored
- **Body:** Georgia serif, 15px, generous leading (1.85-1.9)
- **Section headers:** 10px, uppercase, extreme tracking, neutral-600

### Layout
- **Mobile:** Stacked — hero → title → brief → details
- **Desktop:** Magazine spread — could split to two-column layout
- **Body text:** CSS columns (2-col) on desktop for magazine feel

### Details to Add
- Gold/accent horizontal rules framing the title (Modern Elegance reference)
- Pull quotes from the design brief ("I love rugs..." energy)
- Bulldog clip indicator on pinned lookbook cards
- Multi-image detail grid for iteration results (Positano spread)
- Script/display font option for design name (Meticulous reference)

### Color
- Dark neutral-950 background
- Accent color derived from design palette (mid-range swatch)
- Used sparingly: category label, key move numbers

### Interaction
- Parallax hero with scroll-linked scale + opacity
- Scroll-reveal sections (once, with upward slide)
- Bulldog clip indicator for pinned/starred cards in lookbook
